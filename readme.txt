Import your instagram pics or pics from your instagram feed into individual posts

== Description ==

Import your instagram pics, or pics from your instagram feed, into individual posts. 

Features:
- Import posts and auto publish or save as a draft
- Pick and choose which items to import
- Import posts from your feed (ie, posts from the folks you follow)
- Import hashtags as Wordpress tags
- Import to a specific category
- Easy configurable HTML post template

Visit <a href="http://www.transcendevelopment.com">www.TranscenDevelopment.com</a> for more information.

== Installation ==

1. Upload the "td-instagram-import" folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a new Client ID on Instagram (see http://www.transcendevelopment.com/td-instagram-importer/help.html)
4. Enter the Client ID & Client Secret in the settings
5. Authorize your blog with Instagram.
6. Have at it
